# Security Policy

## Supported Versions

Security updates target the latest commit on the default branch.

## Reporting a Vulnerability

Do not disclose vulnerabilities, leaked credentials, private crawl targets, or sensitive output data in public issues.

If this project is hosted under your GitHub account, enable GitHub private vulnerability reporting or provide a private contact address in this file before publishing.

## Secrets

This project must be configured with environment variables such as `OPENAI_API_KEY`. Never commit real API keys, tokens, private base URLs, cookies, crawl results containing sensitive data, or local agent/tool settings.
